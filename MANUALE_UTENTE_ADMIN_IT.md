# Manuale Utente Admin (FlashPoint)

Questo manuale e dedicato agli amministratori del sistema FlashPoint.

## 1) Ruolo Admin

L'amministratore puo:
- Gestire punti/ore dei dipendenti.
- Gestire profili utente.
- Gestire locali/aziende.
- Gestire richieste.
- Effettuare backup e pulizia dati.
- Analizzare statistiche ed esportare report PDF.

## 2) Menu Amministrazione

Nel menu Admin trovi le aree:
- Gestione ore registrate (`Admin Punti`)
- Statistiche (`Admin Statistiche`)
- Backup e pulizia
- Gestione locali
- Profili utenti
- Richieste dipendenti

## 3) Admin Punti

Funzioni:
- Filtrare per dipendente, mese e locale.
- Visualizzare totale ore in `HH:MM`.
- Modificare o eliminare singoli punti.
- Esportare in PDF i punti del dipendente selezionato.

Nota operativa:
- L'export PDF richiede un dipendente selezionato.
- Note lunghe vengono mandate a capo nel PDF.
- Il PDF include doppia firma: collaboratore e responsabile azienda.

## 4) Backup e Pulizia

Funzioni:
- Scaricare backup CSV per periodo.
- Scaricare backup e cancellare i record dal server.

Attenzione:
- Il backup viene salvato sul PC locale.
- L'azione di cancellazione e irreversibile.
- Usare solo con finestra temporale e filtri corretti.

## 5) Profili Utenti

Funzioni:
- Consultare contatti utente.
- Modificare qualsiasi campo profilo.
- Aprire chat WhatsApp dal numero principale.
- Aprire tesserino del singolo dipendente.

Gestione tesserino:
- Campo etichetta azienda:
- `Distaccato Presso`
- `Datore di Lavoro`

## 6) Gestione Locali

Per ogni locale e possibile creare/modificare:
- Ragione Sociale
- Responsabile tecnico
- Telefono
- Indirizzo
- P.IVA

Funzioni:
- Inserimento nuovo locale.
- Modifica locale esistente.
- Eliminazione locale.

## 7) Admin Richieste

Funzioni:
- Visualizzare tutte le richieste.
- Leggere messaggi mantenendo le interruzioni di riga.
- Aggiornare stato:
- Approvato
- Rifiutato
- Evaso
- Eliminare richieste non valide/duplicate.

## 8) Admin Statistiche

Analisi disponibili:
- Ore per mese/trimestre/semestre/anno.
- Ore del periodo selezionato.
- Ore complessive annue.
- Top localita.
- Grafici ore per localita e per mese.

Export:
- Esportazione statistiche in PDF con riepilogo tabellare.

## 9) Procedure Consigliate

Fine settimana:
- Verifica richieste in sospeso.
- Verifica anomalie ore per locale.

Fine mese:
- Esporta report PDF per dipendente.
- Esegui backup periodo.
- Solo dopo verifica, eventuale pulizia dati.

## 10) Controlli di Qualita Dati

Prima di confermare operazioni sensibili:
- Dipendente corretto.
- Periodo corretto (`MM/AAAA` o intervallo previsto).
- Locale corretto.
- Nessun duplicato evidente.
- Backup eseguito prima di cancellazioni massive.
